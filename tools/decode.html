etchit backup decoder

Decrypts an etchit private-etch backup locally in your browser. The file and password never leave this page.

How to recover the actual etch content

The decoded JSON is an array of entries with fields dm (hex datamap), t (title), ts (timestamp ms). To fetch the encrypted content off the Autonomi network, use the ant CLI:

echo -n '<hex from dm field>' | xxd -r -p > datamap.bin
ant file download --output etch-content datamap.bin

That downloads the self-encrypted chunks and decrypts them locally using the datamap. The result is the original envelope JSON with your etch's title and content.

Works fully offline. You can save this file and open it from disk (file://…) — it needs no network.
